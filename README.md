EcoFriends 🌿

AI Smart Plantation Assistant

Overview

EcoFriends is an AI-powered smart plantation assistant developed for students and beginners who want guidance in growing and maintaining plants.

The app helps users:

Recommend suitable plants
Learn plant care
Detect plant diseases
Predict plant growth
Ask plant-related questions using an AI chatbot

The application is built using:

Python
Streamlit
JavaScript
HTML/CSS
Features
🌱 Plant Recommendation

Recommends plants based on:

Sunlight availability
Watering preference
Climate conditions
🤖 AI Chatbot

Answers plant-related questions such as:

Watering tips
Sunlight requirements
Fertilizer suggestions
Disease guidance
🍂 Leaf Disease Detection

Users can upload leaf images to detect possible plant diseases.

📈 Growth Prediction

Predicts estimated plant growth based on the number of days.

🔐 Login & Create Account

Includes:

User Login
Create Account
Session Management
✨ Interactive UI

Includes:

JavaScript popup welcome message
Clickable navigation
Modern responsive interface
